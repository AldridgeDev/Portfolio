The Basics

The website for Windsor Commercial is built entirely on vanilla php and MySQL to meet customization necessities. Following an agile approach, we constructed a template to follow for the frontend and finished with the backend. The easy to follow CMS is used by other employers to add on to the website without knowing code.

The Details

The beginning stages of the project composed entirely of creating a template for the website. We used the design layout from Windsor's Home website. All styles are done in CSS 3 and is mobile friendly.

The second stage was building backend components. The ability to see all construction projects or only certain construction projects are handled with database queries. These queries allow employees who have access to the CMS to Add, Edit, Delete, and Archive construction projects from the website.

  • Adding allows CMS users to create a new construction project with descriptions and images
  • Editing allows CMS users to change current projects on display or that have been archived
  • Deleting completely removes project access from the CMS user, but still remains in the database and can only be completely removed from the database by an admin
  • Archiving CMS content allows for upcoming construction projects to be written but not posted, or for projects to be removed but accessable for a future return
