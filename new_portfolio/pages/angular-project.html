Angular 2

The Basics

Angular 2 is my current go-to framework. The project I built was focused on using as many tools as possible that angular has to offer. The project's capabilities include creating and editing dynamic document information blocks, messages, contacts, and searching through contacts. Everything can be done without having to send information back to the server and wait for a response, which is frankly one of my favorite features Angular 2.
